# 🖤 JAHOILR — Landing Page Personal

Landing page personal minimalista para **JAHOILR**, desarrollada con HTML, CSS y JavaScript puro. Sin frameworks, sin dependencias externas.

## ✨ Características

- **Diseño dark minimalista** con paleta negro + verde lima (`#c8f135`)
- **Cursor personalizado** con anillo animado y efecto hover
- **Animaciones de entrada** (Intersection Observer API)
- **Contador animado** en la sección de estadísticas
- **Efecto typing** en el hero
- **Navbar** con efecto glassmorphism al hacer scroll
- **100% responsive** para móvil y desktop
- **Sin dependencias** — HTML + CSS + JS puro

## 📁 Estructura

```
landing-jahoilr/
└── index.html    # Todo en un solo archivo (826 líneas)
```

## 🚀 Uso

Simplemente abre `index.html` en cualquier navegador. No requiere servidor ni build.

## 🛠️ Stack

- HTML5 semántico
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- JavaScript ES6+ (IntersectionObserver, requestAnimationFrame)

## 📄 Secciones

1. **Hero** — Presentación con typing effect y CTA
2. **About** — Bio + estadísticas animadas
3. **Skills** — Stack tecnológico agrupado por categoría
4. **Projects** — Tarjetas de proyectos destacados
5. **Contact** — Links de contacto (Email, GitHub, LinkedIn, Twitter)
